## 📖 Описание

**BotPlayer** — это плагин, добавляющий на сервер ботов со скинами, которые ведут себя как настоящие игроки. Боты отображаются как полноценные сущности в мире, пишут сообщения в чат, показываются в **query** и в **таб-листе** (списке игроков), создавая ощущение живого и населённого сервера.

Идеально подходит для того, чтобы:
- 📈 Поднять онлайн сервера в query
- 💬 Оживить чат активностью
- 👥 Заполнить таб-лист «игроками»
- 🎭 Создать атмосферу активного сервера

---

## ✨ Возможности

- 🎨 **Скины** — каждый бот отображается с собственным скином
- 🧍 **Сущности** — боты видны в мире как настоящие игроки
- 💬 **Чат** — боты пишут сообщения в общий чат
- 📊 **Query** — учитываются в онлайне сервера
- 📋 **Таб-лист** — отображаются в списке игроков (TAB)
- ⚙️ **Гибкая настройка** — конфигурация под ваш сервер

## 📦 Установка

1. Скачайте `.jar` файл плагина из репозитория
2. Переместите его в папку `plugins/` вашего сервера
3. Перезапустите сервер
4. Настройте конфигурацию под себя
5. Готово — боты появятся на сервере 🎉

---

## ⚙️ Конфигурация

После первого запуска в папке плагина появится файл конфигурации, где можно настроить:

- Количество ботов
- Скины
- Ники ботов
- Частоту и содержание сообщений в чате
- Отображение в табе и query
