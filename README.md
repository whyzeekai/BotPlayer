## ☠ О плагине

**BotPlayer** — премиум-решение для оживления сервера. Плагин добавляет ботов, которые выглядят и ведут себя как настоящие игроки: у каждого свой скин, они отображаются как сущности в мире, общаются в чате, светятся в **query** и заполняют **таб-лист**.

Больше никаких пустых серверов. Только атмосфера живого проекта `24/7`.

<img src="https://raw.githubusercontent.com/Anmol-Baranwal/Cool-GIFs-For-GitHub/main/assets/rainbow_line_neon.gif" width="100%"/>

## ⚡ Что умеет

> 🎨 **Скины** — каждый бот со своим внешним видом
>
> 🧍 **Сущности** — боты реально присутствуют в мире
>
> 💬 **Чат** — живое общение и активность
>
> 📡 **Query** — учитываются в реальном онлайне
>
> 📋 **Таб-лист** — красиво отображаются в списке игроков
>
> ⚙️ **Гибкая настройка** — полный контроль под твой проект
