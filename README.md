## 🌌 О плагине

**BotPlayer** — это премиум-решение для оживления сервера. Плагин добавляет ботов, которые выглядят и ведут себя как настоящие игроки: у каждого свой скин, они отображаются как сущности в мире, общаются в чате, светятся в **query** и заполняют **таб-лист**.

Больше никаких пустых серверов. Только атмосфера живого, активного проекта.

<br>

## ⚡ Что умеет

> 🎨 **Скины** — каждый бот со своим внешним видом
>
> 🧍 **Сущности** — боты реально присутствуют в мире
>
> 💬 **Чат** — живое общение и активность в чате
>
> 📡 **Query** — учитываются в реальном онлайне сервера
>
> 📋 **Таб-лист** — красиво отображаются в списке игроков
>
> ⚙️ **Гибкая настройка** — полный контроль под твой проект
